Introduction

Base URL: https://wade.monster.com

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

註冊email驗證信確認

電子郵件驗證確認

此端點用於確認用戶的電子郵件驗證。它會比對提供的hash值和用戶的電子郵件生成的hash值。 如果驗證成功，該用戶的電子郵件將被標記為已驗證，並且將觸發一個已驗證的事件。

Register

Operations related to register.

處理新使用者的註冊並寄送註冊信。

此方法會驗證輸入的資料，並在成功驗證後在users表中創建一個新的使用者紀錄。 之後，它會觸發一個Registered事件，並返回一個成功的響應，包括新創建的使用者的資料。

Example request:
curl --request POST \
    "https://wade.monster.com/api/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"corporis\",
    \"email\": \"abartoletti@example.org\",
    \"password\": \"\\\\Kzs\\\"*~1:3V2C?<KRl\",
    \"password_confirmation\": \"sed\"
}"

const url = new URL(
    "https://wade.monster.com/api/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "corporis",
    "email": "abartoletti@example.org",
    "password": "\\Kzs\"*~1:3V2C?<KRl",
    "password_confirmation": "sed"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

此方法允許更新使用者的基本資訊。如果請求中包含用戶照片，它將生成一個預簽名的URL， 以便用戶的瀏覽器可以直接上傳圖片到S3。

curl --request POST \
    "https://wade.monster.com/api/user" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "name=dolorem"\
    --form "email=lreinger@example.net"\
    --form "userPhoto=@/private/var/folders/fj/8mc653gd5ms9d4bnjf78tjtm0000gn/T/php85fHGW" 
const url = new URL(
    "https://wade.monster.com/api/user"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('name', 'dolorem');
body.append('email', 'lreinger@example.net');
body.append('userPhoto', document.querySelector('input[name="userPhoto"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());
